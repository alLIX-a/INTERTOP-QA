# BUG-003: Сброс пароля — установка текущего пароля в качестве нового разрешена
| Поле | Детали |
|---|---|
| **ID** | BUG-003 |
| **Серьёзность** | Средняя |
| **Статус** | К выполнению |
| **Репортер** | Alexey |
| **Связанный ТК** | [TC-005](../test-cases/TC-005-resetting-password-via-email-for-existing-user.md) |
## Предусловия
1. Главная страница открыта
2. Пользователь зарегистрирован с данными из раздела Тестовые данные данного ТК, по сценарию [TC-001](../test-cases/TC-001-signing-up-with-valid-email.md)
3. Пользователь не авторизован
4. Почтовый ящик для email из раздела Тестовые данные открыт в любом почтовом сервисе
## Тестовые данные
| Поле | Значение |
|---|---|
| Email | visiwi6199@imashr.com (TempMail) |
| Пароль от email | Test123 |
| Пароль от аккаунта | Test123 |
## Шаги для воспроизведения
1. Нажать на иконку персоны в правом верхнем углу для открытия формы
2. Вставить email в поле "Телефон или email"
3. Нажать на кнопку "Забыли пароль?"
4. Появляется новое поле для email, на который будет отправлено письмо для сброса пароля. Поле автоматически заполняется email, введённым в поле "Телефон или Email". Проверить совпадение между ними
5. Нажать на кнопку "ОТПРАВИТЬ"
6. Нажать на чекбокс "I'm not a robot" и пройти капчу
7. Нажать на кнопку "ОТПРАВИТЬ" повторно
8. Открыть письмо для восстановления доступа в почтовом ящике
9. В открытом письме нажать на кнопку "ИЗМЕНИТЬ МОЙ ПАРОЛЬ"
10. В открывшейся новой вкладке вставить пароль от аккаунта в поле "Новый пароль"
11. Нажать на кнопку "ОТПРАВИТЬ"
## Ожидаемый результат
Появляется сообщение об ошибке с требованием ввести пароль, отличающийся от текущего.
## Фактический результат
Появилось окно с сообщением "Пароль успешно изменён!" и пароль был успешно изменён на такой же без каких-либо ошибок валидации.
