# BUG-009: Поле поиска — кнопка очистки поля поиска не работает

| Поле | Значение |
|---|---|
| **ID** | BUG-009 |
| **Серьезность** | Средняя |
| **Приоритет** | Средний |

## Предусловия

1. Пользователь находится на любой странице веб-сайта
2. Пользователь может быть авторизован или нет

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название товара | кеды vans |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить название товара
3. Нажать на иконку крестика внутри поля поиска

## Ожидаемый результат

Текст внутри поля поиска удаляется.

## Фактический результат

Текст внутри поля поиска не удалился, при нажатии на иконку крестика ничего не происходит. При инспекции элемента в DevTools собственных обработчиков элемента в Event Listeners найдено не было. При включении чекбокса Ancestors среди click-обработчиков родительских элементов обработчик очистки поля не обнаружен.

## Environment

| Параметр | Значение |
|---|---|
| ОС | macOS Tahoe 26.4 |
| Браузер | Safari 26.4 (21624.1.16.11.4) |
