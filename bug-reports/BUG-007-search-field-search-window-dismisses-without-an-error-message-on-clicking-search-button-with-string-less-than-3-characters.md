# BUG-007: Поле поиска — окно с полем закрывается без сообщения об ошибке после нажатия на кнопку поиска при длине запроса меньше 3 символов

| Поле | Значение |
|---|---|
| **ID** | BUG-007 |
| **Severity** | Medium |
| **Priority** | Medium |

## Предусловия

1. [Главная страница](https://intertop.kz/ru-kz/) открыта
2. Статус авторизации не имеет значения

## Тестовые данные

| Поле | Значение |
|---|---|
| Название товара | ab |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить название товара
3. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Показывается сообщение об ошибке, указывающее на то что минимальная длина запроса должна составлять 3 символа. Окно с полем поиска не закрывается.

## Фактический результат

Окно с полем поиска закрывается без сообщения об ошибке.
