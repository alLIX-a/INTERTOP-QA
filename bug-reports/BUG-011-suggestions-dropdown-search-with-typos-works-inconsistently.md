# BUG-011: Выпадающее меню предложений — поиск с опечатками работает непоследовательно
| Поле | Значение |
|---|---|
| **ID** | BUG-011 |
| **Серьезность** | Средняя |
| **Приоритет** | Средний |
## Предусловия
1. Пользователь находится на любой странице веб-сайта
2. Пользователь может быть авторизован или нет
## Тестовые данные
| Параметр | Значение |
|---|---|
| Название запроса | vanz |
## Шаги воспроизведения
1. Нажать на кнопку "Поиск модных находок"
2. Вставить поисковый запрос из раздела "Тестовые данных" в поле поиска
## Ожидаемый результат
Появляется выпадающее меню, содержащее до 10 релевантных предложений, наиболее близких к поисковому запросу. Система корректирует опечатку в списке предложений и отображает результаты для схожих по написанию слов (аналогично запросу "adidaz", который корректируется до "adidas").
## Фактический результат
Выпадающее меню с релевантными предложениями не появилось.
## Environment
| Параметр | Значение |
|---|---|
| ОС | macOS Tahoe 26.4 |
| Браузер | Safari 26.4 (21624.1.16.11.4) |
