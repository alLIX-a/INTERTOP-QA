# BUG-006: Поле поиска — окно с полем закрывается без сообщения об ошибке после нажатия на кнопку поиска при пустом поле

| Поле | Детали |
|---|---|
| **ID** | BUG-006 |
| **Серьёзность** | Низкая |
| **Приоритет** | Низкий |
| **Статус** | К выполнению |

## Предусловия

1. [Главная страница](https://intertop.kz/ru-kz/) открыта
2. Статус авторизации не имеет значения

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. Нажать на иконку лупы внутри появившегося поля поиска

## Ожидаемый результат

Показывается сообщение об ошибке, предлагая пользователю ввести название интересующего товара. Товаров по запросу не найдено.

## Фактический результат

Окно с полем поиска закрылось после нажатия на иконку лупы, сообщение об ошибке не отобразилось. Товаров по запросу не найдено.

| Поле | Значение |
|---|---|

| Браузер | Safari 26.4 (21624.1.16.11.4) |
