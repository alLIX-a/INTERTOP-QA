# BUG-014: Фильтрация товаров по цене — система не отображает сообщение об ошибке при минимальной цене выше максимальной

| Поле | Значение |
|---|---|
| **ID** | BUG-014 |
| **Серьезность** | Низкая |
| **Приоритет** | Низкий |

## Предусловия

1. Страница товаров открыта
2. Пользователь может быть авторизован или нет

## Тестовые данные

1. Минимальная цена: максимальная цена + 1 ₸ (актуальный максимум на момент написания ТС — 395 990 ₸, может меняться. Максимальная цена указана в выпадающем списке фильтра цен в правом поле меню)

## Шаги воспроизведения

1. Нажать на кнопку "Цена"
2. В выпадающем меню ввести значение на единицу выше максимального в левое поле меню
3. Нажать на кнопку "Ок"

## Ожидаемый результат

Система предотвращает применение некорректного диапазона.

Допустимый вариант поведения 1: система автоматически корректирует значение нижней границы до значения максимальной или максимум − 1.

Допустимый вариант поведения 2: система не применяет фильтр и показывает сообщение об ошибке.

Недопустимо: применение фильтра с минимальной границей выше максимальной, отображение некорректных или пустых результатов без объяснения.

## Фактический результат

Фильтр не применён, сообщение об ошибке показано не было.
