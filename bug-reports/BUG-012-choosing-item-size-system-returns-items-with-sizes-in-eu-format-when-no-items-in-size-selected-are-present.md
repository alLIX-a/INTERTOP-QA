# BUG-012: Выбор размера товара — при отсутствии товаров в заданном размере система возвращает товары в другой размерной сетке

| Поле | Значение |
|---|---|
| **ID** | BUG-012 |
| **Серьезность** | Средняя |
| **Приоритет** | Средний |

## Предусловия

1. Страница товаров открыта
2. Пользователь может быть авторизован или нет

## Тестовые данные

| Параметр | Значение |
|---|---|
| Международный размер | 4XL |

## Шаги воспроизведения

1. Нажать на кнопку "Размер одежды"
2. В выпадающем списке размеров выбрать размер 4XL

## Ожидаемый результат

При отсутствии товаров в размере 4XL отображается пустая выдача или сообщение "товары не найдены".

## Фактический результат

Отобразились товары, на карточке которых размер указан в европейском стандарте. API-запрос при этом корректно передаёт размер 4XL как параметр запроса.
