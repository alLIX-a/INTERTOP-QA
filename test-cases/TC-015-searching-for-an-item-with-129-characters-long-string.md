# TC-015: Поиск товара со строкой в 129 символов (на 1 больше максимальной границы)

| Поле | Значение |
|---|---|
| **ID** | TC-015 |
| **Название** | Поиск товара со строкой в 129 символов (на 1 больше максимальной границы) |

## Предусловия

1. Главная страница открыта
2. Пользователь может быть авторизован или нет
3. Случайная строка из 129 символов, сгенерированная на texttools.ru

## Тестовые данные

| Поле | Значение |
|---|---|
| Данные для ввода | Случайно сгенерированная строка из 129 символов |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить сгенерированную строку
3. Открыть DevTools
4. Инспектировать строку внутри поля в DevTools
5. Найти у элемента атрибут `maxlength`
6. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Длина сгенерированной строки урезается до 128 символов после вставки в поле поиска, в соответствии со значением атрибута `maxlength` = 128. Отображается сообщение "По вашему запросу ничего не найдено. Попытайтесь изменить запрос или выбрать другой раздел для уточнения". Ошибок нет. Товаров по запросу не найдено.

## Фактический результат

Длина сгенерированной строки урезалась до 128 символов после вставки в поле поиска, `maxlength` = 128. Ожидаемое сообщение отобразилось. Ошибок нет, товаров по запросу не найдено.
