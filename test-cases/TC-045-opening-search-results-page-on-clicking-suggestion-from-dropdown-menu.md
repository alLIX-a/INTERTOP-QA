# TC-045: Переход на страницу результатов поиска при выборе предложения из выпадающего меню
| Поле | Значение |
|---|---|
| **ID** | TC-045 |
| **Название** | Переход на страницу результатов поиска при выборе предложения из выпадающего меню |
## Предусловия
1. Пользователь находится на любой странице веб-сайта
2. Пользователь может быть авторизован или нет
## Тестовые данные
| Параметр | Значение |
|---|---|
| Название запроса | vans |
## Шаги воспроизведения
1. Нажать на кнопку "Поиск модных находок"
2. Вставить поисковый запрос из раздела "Тестовые данные" в поле поиска
3. В появившемся выпадающем меню нажать на любое из предложений
## Ожидаемый результат
Открывается новая страница в текущей вкладке, показываются результаты поиска. Результаты частично и/или полностью соответствуют выбранному предложению в выпадающем меню. Результаты отсортированы по популярности.
## Фактический результат
Открылась новая страница в текущей вкладке, показанные результаты поиска соответствуют выбранному предложению и отсортированы по популярности.
## Статус
**Пройден**
