# TC-027: Добавление товара отсутствующим в наличии размером, выбранным размером из окна выбора размера

| Поле | Значение |
|---|---|
| **ID** | TC-027 |
| **Название** | Добавление товара отсутствующим в наличии размером, выбранным размером из окна выбора размера |

## Предусловия

1. Страница товара открыта
2. Пользователь может быть авторизован или нет
3. Товар имеет хотя бы один размер, отсутствующий в наличии
4. Раздел "Корзина" очищен во избежание конфликта цены, размера и/или количества

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название товара | Кеды Vans |
| Цвет | Черный (выбран системой автоматически) |
| Размер | Любой из отсутствующих в наличии размеров в окне "Выберите размер" |
| Количество | 1 (установлен системой автоматически) |

## Шаги воспроизведения

1. Нажать на кнопку "Добавить в корзину"
2. В появившемся окне "Выберите размер", выбрать любой размер отсутствующий в наличии (у размеров отсутствующих в наличии присутствует кнопка "Сообщить" напротив размера)
3. Нажать на кнопку "Добавить в корзину" в окне
4. Закрыть окно нажав на иконку крестика в правом верхнем углу
5. Перейти в раздел "Корзина", нажав на иконку корзины в правом верхнем углу страницы

## Ожидаемый результат

При нажатии на кнопку "Добавить в корзину" после выбора отсутствующего в наличии размера ничего не происходит. Товар в разделе "Корзина" отсутствует.

## Фактический результат

При нажатии на кнопку "Добавить в корзину" товар в корзину добавлен не был.

## Статус

**Пройден**
