# TC-024: Добавление товара с не заданным размером в корзину

| Поле | Значение |
|---|---|
| **ID** | TC-024 |
| **Название** | Добавление товара с не заданным размером в корзину |

## Предусловия

1. Страница товара открыта
2. Товар есть в наличии в любом из размеров
3. Раздел "Корзина" очищен во избежание конфликта цены, размера и/или количества

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название товара | Кеды Vans |
| Цвет | Черный (выбран системой автоматически) |
| Количество | 1 (установлен системой автоматически) |

## Шаги воспроизведения

1. Нажать на кнопку "Добавить в корзину"
2. Закрыть появившееся окно "Выберите размер", нажав на иконку крестика в правом верхнем углу окна
3. Открыть раздел "Корзина", нажав на иконку корзины в правом верхнем углу страницы

## Ожидаемый результат

Появляется окно "Выберите размер" при нажатии на кнопку "Добавить в корзину", товар в разделе "Корзина отсутствует"

## Фактический результат

При добавлении товара в корзину появилось окно "Выберите размер", товар в корзину добавлен не был.

## Статус

**Пройдено**
