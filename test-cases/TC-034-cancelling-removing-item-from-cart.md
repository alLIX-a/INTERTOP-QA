# TC-034: Отмена удаления товара из корзины

| Поле | Значение |
|---|---|
| **ID** | TC-034 |
| **Название** | Отмена удаления товара из корзины |

## Предусловия

1. Пользователь находится на любой странице сайта
2. Пользователь может быть авторизован или нет
3. В корзину добавлен хотя бы 1 товар (см. TC-023)

## Тестовые данные

| Параметр | Значение |
|---|---|
| Товар | Любой товар, добавленный в корзину |

## Шаги воспроизведения

1. Открыть раздел "Корзина", нажав на иконку корзины в правом верхнем углу страницы
2. Нажать на иконку мусорного ведра напротив любого из товаров в корзине. В появившемся окне будет запущен таймер в 3 секунды перед удалением товара. Следующий шаг необходимо выполнить в течение этих 3 секунд
3. В появившемся окне "Товар удален из корзины", нажать на "Отменить удаление"

## Ожидаемый результат

Окно "Товар удален из корзины" пропадает, карточка товара снова видна в корзине. Товар не удалён по истечении 3 секунд.

## Фактический результат

Окно пропало, карточка товара обратно появилась в корзине. Товар не удалён.

## Статус

**Пройден**
