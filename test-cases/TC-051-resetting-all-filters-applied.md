# TC-051: Сброс всех применённых фильтров

| Поле | Значение |
|---|---|
| **ID** | TC-051 |
| **Название** | Сброс всех применённых фильтров |

## Предусловия

1. Страница товаров открыта
2. Пользователь может быть авторизован или нет
3. Применено хотя бы 2 любых фильтра (см. TC-049)

## Тестовые данные

| Параметр | Значение |
|---|---|
| Фильтры | 2 применённых фильтра, полученных как результат выполнения TC-049 |

## Шаги воспроизведения

1. Нажать на кнопку "Очистить все фильтры", находящуюся под списком фильтров

## Ожидаемый результат

Список применённых фильтров очищен, страница товаров отображает весь каталог без ограничений по фильтрам.
