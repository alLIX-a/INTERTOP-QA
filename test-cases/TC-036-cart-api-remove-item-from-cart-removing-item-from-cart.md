# TC-036: cartApi/removeItemFromCart - удаление товара из корзины

| Поле | Значение |
|---|---|
| **ID** | TC-036 |
| **Название** | cartApi/removeItemFromCart - удаление товара из корзины |

## Предусловия

1. Postman открыт
2. Пользователь авторизован (см. TC-002)
3. Выполнить TC-033. После выполнения открыть DevTools → Network → removeItemFromCart → вкладка Cookies. Во вкладке скопировать значения userV2 и cartV3

## Тестовые данные

| Параметр | Значение |
|---|---|
| Method | POST |
| Endpoint | https://front-api.intertop.kz/cartApi/removeItemFromCart |
| X-locale | ru-kz |
| X-site | kz |
| Cookies | Актуальные значения userV2 и cartV3 |
| Тело запроса | `{ "id": "9115926911594249690", "offerId": "9115942" }` |

## Шаги воспроизведения

1. Создать запрос по указанному endpoint
2. Задать X-locale в заголовке запроса
3. Задать X-site в заголовке запроса
4. Задать Cookie в заголовке запроса следующим образом: userV2=значение;cartV3=значение
5. Нажать на кнопку "Send"

## Ожидаемый результат

Сервер возвращает ответ со статус кодом 200 OK и пустым телом.

## Фактический результат

Тест заблокирован: TC-002 провален, авторизация через email недоступна, получение cookie userV2 невозможно.

## Статус
