# TC-028: Добавление товара в корзину с измененным размером перед добавлением

| Поле | Значение |
|---|---|
| **ID** | TC-028 |
| **Название** | Добавление товара в корзину с измененным размером перед добавлением |

## Предусловия

1. Страница товара открыта
2. Пользователь может быть авторизован или нет
3. Раздел "Корзина" очищен во избежание конфликта цены, размера и/или количества
4. Товар доступен хотя бы в 2 размерах

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название товара | Кеды Vans |
| Цвет | Черный (выбран системой автоматически) |
| Количество | 1 (установлен системой автоматически) |
| Размер | Любые 2 размера в наличии (размеры в наличии выделены серым цветом) |

## Шаги воспроизведения

1. Выбрать любой из доступных размеров в разделе "Размер"
2. Выбрать любой другой размер в наличии
3. Нажать на кнопку "Добавить в корзину"
4. В появившемся окне "Товар успешно добавлен в корзину", нажать на кнопку "В корзину"

## Ожидаемый результат

Добавленный товар появляется в корзине. Количество товара в корзине соответствует выбранному. Размер товара в корзине соответствует последнему выбранному размеру на странице товара. Цена товара в корзине соответствует цене, указанной на странице товара.

## Фактический результат

Товар появился в корзине. Размер соответствует последнему выбранному. Количество соответствует выбранному. Цена соответствует указанной на странице.

## Статус

**Пройден**
