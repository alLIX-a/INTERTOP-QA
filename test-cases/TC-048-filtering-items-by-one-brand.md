# TC-048: Фильтрация товаров по одному бренду

| Поле | Значение |
|---|---|
| **ID** | TC-048 |
| **Название** | Фильтрация товаров по одному бренду |

## Предусловия

1. Страница товаров открыта
2. Пользователь может быть авторизован или нет

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название бренда | Adidas |

## Шаги воспроизведения

1. Нажать на кнопку "Бренд"
2. В выпадающем списке брендов найти и выбрать бренд Adidas

## Ожидаемый результат

Отображаются все и только товары бренда Adidas.

## Фактический результат

Товары только бренда Adidas отобразились и отсортированы по популярности.

## Статус

**Пройден**
