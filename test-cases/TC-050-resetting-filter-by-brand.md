# TC-050: Сброс фильтра товаров по бренду

| Поле | Значение |
|---|---|
| **ID** | TC-050 |
| **Название** | Сброс фильтра товаров по бренду |

## Предусловия

1. [Страница товаров](ССЫЛКА) открыта
2. Пользователь может быть авторизован или нет
3. Применён хотя бы 1 фильтр по бренду (см. TC-049)

## Тестовые данные

| Параметр | Значение |
|---|---|
| Фильтр | Фильтр, применённый как результат выполнения TC-049 |

## Шаги воспроизведения

1. Нажать на иконку крестика напротив применённого фильтра, находящегося под списком фильтров

## Ожидаемый результат

Список применённых фильтров очищен, страница товаров отображает весь каталог без ограничений по бренду.

## Фактический результат

Список применённых фильтров очистился, страница товаров отображает весь каталог без ограничений по бренду.

## Статус

**Пройден**
