# TC-040: Сохранение в историю поиска товара существующего в истории в верхнем регистре

| Поле | Значение |
|---|---|
| **ID** | TC-040 |
| **Название** | Сохранение в историю поиска товара существующего в истории в верхнем регистре |

## Предусловия

1. Пользователь находится на любой странице веб-сайта
2. Пользователь может быть авторизован или нет
3. Поиск товара "кеды vans" выполнен (см. TC-011)
4. Товар "кеды vans" присутствует в разделе "История поиска" (см. TC-037)

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название товара | КЕДЫ VANS |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. Стереть ранее осуществлённый поисковой запрос в поле поиска, нажав на иконку крестика внутри поля
3. Повторно выполнить TC-011, используя тестовые данные
4. Повторно нажать на кнопку "Поиск модных находок"
5. Повторно стереть содержимое поля поиска

## Ожидаемый результат

Запрос "КЕДЫ VANS" в истории поиска не появляется. Запрос "кеды vans" сохраняется в истории.

## Фактический результат

Запрос "КЕДЫ VANS" в истории не появился, запрос "кеды vans" сохранился в истории.

## Статус

**Пройден**
