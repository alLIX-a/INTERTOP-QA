# TC-019: Поиск товара с запросом длиной в 2 символа (минимально допустимая — 3 символа)

| Поле | Значение |
|---|---|
| **ID** | TC-019 |
| **Название** | Поиск товара с запросом длиной в 2 символа (минимально допустимая — 3 символа) |
| **Статус** | Провалено |

## Предусловия

1. Главная страница открыта
2. Статус авторизации не имеет значения

## Тестовые данные

| Поле | Значение |
|---|---|
| Название товара | ab |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить название товара
3. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Появляется сообщение об ошибке, требующее от пользователя ввести запрос длиной минимум в 3 символа. Поиск результатов не дает, окно с полем поиска не закрывается.

## Фактический результат

Окно с полем поиска закрылось, сообщение об ошибке не было показано. Поиск результатов не дал
