# TC-030: Добавление товара 2 размеров одновременно в корзину

| Поле | Значение |
|---|---|
| **ID** | TC-030 |
| **Название** | Добавление товара 2 размеров одновременно в корзину |

## Предусловия

1. Страница товара открыта
2. Товар доступен хотя бы в 2 размерах в наличии
3. Пользователь может быть авторизован или нет
4. Раздел "Корзина" очищен во избежание конфликта цены, размера и/или количества

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название товара | Кеды Vans |
| Цвет | Черный (выбран системой автоматически) |
| Размер | Любые 2 размера в наличии |
| Количество | 2, по 1 для каждого из размеров |

## Шаги воспроизведения

1. Нажать на кнопку "Добавить в корзину"
2. В появившемся окне "Выберите размер", выбрать любые 2 размера в наличии (у размеров в наличии отсутствует кнопка "Сообщить" напротив размера)
3. Нажать на кнопку "Добавить в корзину" в окне
4. В появившемся окне "Товар успешно добавлен в корзину", нажать на кнопку "В корзину"

## Ожидаемый результат

Товар обоих выбранных размеров появляется в корзине. Каждый из добавленных размеров является отдельным товаром. Размер и количество для каждого из размеров соответствуют выбранным на странице товара. Цена для каждого размера в корзине соответствует цене на странице товара.

## Фактический результат

Товар появился в корзине в двух разных размерах, размеры в корзине соответствуют выбранным на странице товара, цена для каждого соответствует указанной на странице. Оба размера представляют собой отдельные товары в корзине.

## Статус

**Пройден**
