# TC-039: Сохранение в историю поиска товара существующего в истории

| Поле | Значение |
|---|---|
| **ID** | TC-039 |
| **Название** | Сохранение в историю поиска товара существующего в истории |

## Предусловия

1. Пользователь находится на любой странице веб-сайта
2. Пользователь может быть авторизован или нет
3. Поиск товара "кеды vans" выполнен (см. TC-011)
4. Товар "кеды vans" присутствует в разделе "История поиска" (см. TC-037)

## Тестовые данные

| Параметр | Значение |
|---|---|
| Товар | Осуществлённый ранее поисковой запрос "кеды vans" |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. Стереть ранее осуществлённый поисковой запрос в поле поиска, нажав на иконку крестика внутри поля
3. Повторно осуществить поиск товара, выполнил TC-011
4. Повторно нажать на кнопку "Поиск модных находок"
5. Повторно стереть содержимое поля поиска

## Ожидаемый результат

При повторном поиске товара "кеды vans" запрос в истории поиска не дублируется.

## Фактический результат

При повторном запросе товар в истории не продублировался.

## Статус

**Пройден**
