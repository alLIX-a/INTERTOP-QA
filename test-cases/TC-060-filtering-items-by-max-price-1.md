# TC-060: Фильтрация товаров по цене на единицу ниже максимальной

| Поле | Значение |
|---|---|
| **ID** | TC-060 |
| **Название** | Фильтрация товаров по цене на единицу ниже максимальной |

## Предусловия

1. Страница товаров открыта
2. Пользователь может быть авторизован или нет

## Тестовые данные

1. Цена: текущий максимум фильтра − 1 ₸ (актуальный максимум на момент написания ТС — 395 990 ₸, может меняться. Максимальная цена указана в выпадающем списке фильтра цен в правом поле меню)

## Шаги воспроизведения

1. Нажать на кнопку "Цена"
2. В выпадающем меню ввести значение на единицу ниже максимально допустимого в правое поле меню
3. Нажать на кнопку "Ок"

## Ожидаемый результат

Применяется ценовой фильтр с минимальным (установлен по умолчанию) и заданным максимальным диапазоном. Отображаются все и только товары с ценой не выше заданной. Применённый фильтр отображается под списком доступных фильтров.

## Фактический результат

Ценовой фильтр применился корректно и отображается под списком доступных фильтров. Отобразились все и только товары с ценой не выше заданной.

## Статус

**Пройден**
