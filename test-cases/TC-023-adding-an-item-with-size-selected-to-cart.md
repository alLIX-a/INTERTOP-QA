# TC-023: Добавление товара с заданным размером в корзину

| Поле | Значение |
|---|---|
| **ID** | TC-023 |
| **Название** | Добавление товара с заданным размером в корзину |

## Предусловия

1. Страница товара открыта
2. Товар есть в наличии в любом из размеров
3. Раздел "Корзина" очищен во избежание конфликта цены, размера и/или количества

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название товара | Кеды Vans |
| Размер | Любой размер в наличии (доступные размеры отмечены серым цветом) |
| Цвет | Черный (выбран системой автоматически) |
| Количество | 1 (установлен системой автоматически) |

## Шаги воспроизведения

1. Выбрать любой из доступных размеров в разделе "Размер"
2. Нажать на кнопку "Добавить в корзину"
3. В появившемся окне "Товар успешно добавлен в корзину" нажать на кнопку "В корзину"

## Ожидаемый результат

Добавленный товар появляется в корзине. Количество и размер товара в корзине соответствуют количеству и размеру, выбранным на странице товара. Цена товара в корзине соответствует цене, указанной на странице товара.

## Фактический результат

Добавленный товар появился в корзине. Количество и размер в корзине соответствуют выбранным. Цена в корзине соответствует цене на странице товара.

## Статус

**Пройден**
