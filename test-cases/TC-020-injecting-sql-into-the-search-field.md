# TC-020: Ввод SQL-инъекции в поле поиска товара

| Поле | Значение |
|---|---|
| **ID** | TC-020 |
| **Название** | Ввод SQL-инъекции в поле поиска товара |
| **Статус** | Пройдено |

## Предусловия

1. Главная страница открыта
2. Статус авторизации не имеет значения

## Тестовые данные

| Поле | Значение |
|---|---|
| SQL-инъекция | `' OR '1'='1` |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить SQL-инъекцию
3. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Открывается новая страница в текущей вкладке, пользователю показывается сообщение "По вашему запросу ничего не найдено. Попытайтесь изменить запрос или выбрать другой раздел для уточнения.". Товаров по запросу не найдено.

## Фактический результат

Открылась новая страница в текущей вкладке, ожидаемое сообщение отобразилось. Товаров по запросу не найдено.
