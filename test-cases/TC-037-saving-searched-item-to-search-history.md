# TC-037: Сохранение товара в историю поиска

| Поле | Значение |
|---|---|
| **ID** | TC-037 |
| **Название** | Сохранение товара в историю поиска |

## Предусловия

1. Пользователь находится на любой странице веб-сайта
2. Пользователь может быть авторизован или нет
3. Поиск товара "кеды vans" выполнен (см. TC-011)

## Тестовые данные

| Параметр | Значение |
|---|---|
| Товар | Осуществлённый ранее поисковой запрос "кеды vans" |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. Стереть ранее осуществлённый поисковой запрос в поле поиска, нажав на иконку крестика внутри поля

## Ожидаемый результат

Под поисковым полем появляется раздел "История поиска", название ранее осуществлённого поискового запроса сохраняется в историю поиска.

## Фактический результат

Раздел "История поиска" появился, поисковой запрос в разделе появился. При нажатии на иконку крестика поле поиска не очищается: пользователь может очистить поле только при помощи клавиатуры своего устройства.

## Статус

**Пройден**
