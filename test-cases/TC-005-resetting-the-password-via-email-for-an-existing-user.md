# TC-005: Сброс пароля через email для существующего пользователя
| Поле | Детали |
|---|---|
| **ID** | TC-005 |
| **Название** | Сброс пароля через email для существующего пользователя |
| **Статус** | Заблокирован |
## Предусловия
1. Главная страница открыта
2. Пользователь зарегистрирован с данными из раздела Тестовые данные данного ТК, по сценарию [TC-001](TC-001-signing-up-with-valid-email.md)
3. Пользователь не авторизован
4. Почтовый ящик для email из раздела Тестовые данные открыт в любом почтовом сервисе
## Тестовые данные
| Поле | Значение |
|---|---|
| Email | visiwi6199@imashr.com (TempMail) |
| Пароль от email | Test123 |
| Пароль от аккаунта | Test123 |
> ⚠️ После выполнения данного ТК обновить пароль от аккаунта в разделе Тестовые данные на новый, использованный в шаге 10.
## Шаги
1. Нажать на иконку персоны в правом верхнем углу для открытия формы
2. Вставить email в поле "Телефон или Email"
3. Нажать на кнопку "Забыли пароль?"
4. Появляется новое поле для email, на который будет отправлено письмо для сброса пароля. Поле автоматически заполняется email, введённым в поле "Телефон или Email". Проверить совпадение между ними
5. Нажать на кнопку "ОТПРАВИТЬ"
6. Нажать на чекбокс "I'm not a robot" и пройти капчу
7. Нажать на кнопку "ОТПРАВИТЬ" повторно
8. Открыть письмо для восстановления доступа в почтовом ящике
9. В открытом письме нажать на кнопку "ИЗМЕНИТЬ МОЙ ПАРОЛЬ"
10. В открывшейся новой вкладке ввести новый пароль длиной не менее 6 символов в поле "Новый пароль"
11. Нажать на кнопку "ОТПРАВИТЬ"
## Ожидаемый результат
Появляется окно с сообщением "Пароль успешно изменён!", которое автоматически закрывается примерно через 3 секунды. Пользователь может войти с новым паролем. Проверить поведение по сценарию [TC-002](TC-002-signing-in-with-existing-email.md).
## Фактический результат
Окно с сообщением "Пароль успешно изменён!" появилось и автоматически закрылось примерно через 3 секунды. Войти не удалось — после ввода валидных данных для завершения авторизации требуется привязка номера телефона.
