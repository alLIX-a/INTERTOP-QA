# TC-041: Удаление товара из истории поиска

| Поле | Значение |
|---|---|
| **ID** | TC-041 |
| **Название** | Удаление товара из истории поиска |

## Предусловия

1. Пользователь находится на любой странице веб-сайта
2. Пользователь может быть авторизован или нет
3. Поиск товара "кеды vans" выполнен (см. TC-011)
4. Товар "кеды vans" присутствует в разделе "История поиска" (см. TC-037)

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название сохранённого запроса | кеды vans |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. Стереть ранее осуществлённый поисковой запрос в поле поиска, нажав на иконку крестика внутри поля
3. Удалить запрос "кеды vans" из истории поиска, нажав на иконку крестика напротив запроса

## Ожидаемый результат

Запрос "кеды vans" пропадает из истории поиска.

## Фактический результат

Запрос "кеды vans" пропал из истории поиска.

## Статус

**Пройден**
