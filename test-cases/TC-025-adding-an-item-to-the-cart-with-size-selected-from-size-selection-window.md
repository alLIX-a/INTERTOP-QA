# TC-025: Добавление товара с выбранным размером из окна выбора размера

| Поле | Значение |
|---|---|
| **ID** | TC-025 |
| **Название** | Добавление товара с выбранным размером из окна выбора размера |

## Предусловия

1. Страница товара открыта
2. Товар есть в наличии в любом из размеров
3. Раздел "Корзина" очищен во избежание конфликта цены, размера и/или количества
4. Пользователь может быть авторизован или нет

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название товара | Кеды Vans |
| Цвет | Черный (выбран системой автоматически) |
| Количество | 1 (установлен системой автоматически) |
| Размер | Любой из доступных размеров в окне "Выберите размер" |

## Шаги воспроизведения

1. Нажать на кнопку "Добавить в корзину"
2. В появившемся окне "Выберите размер", выбрать любой размер в наличии (у размеров в наличии отсутствует кнопка "Сообщить" напротив размера)
3. Нажать на кнопку "Добавить в корзину" в окне
4. В появившемся окне "Товар успешно добавлен в корзину", нажать на кнопку "В корзину"

## Ожидаемый результат

Добавленный товар появляется в корзине. Количество товара в корзине соответствует количеству, выбранному на странице товара. Размер в корзине соответствует размеру, выбранному в окне выбора размера. Цена товара в корзине соответствует цене, указанной на странице товара.

## Фактический результат

Добавленный товар появился в корзине. Количество и размер в корзине соответствуют выбранным. Цена в корзине соответствует цене на странице товара.

## Статус

**Пройден**
