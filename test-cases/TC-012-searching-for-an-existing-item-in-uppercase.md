# TC-012: Поиск существующего товара в верхнем регистре

| Поле | Значение |
|---|---|
| **ID** | TC-012 |
| **Название** | Поиск существующего товара в верхнем регистре |
| **Статус** | Пройдено |

## Предусловия

1. Главная страница открыта
2. Статус авторизации не имеет значения
3. Кеды бренда "Vans" существуют в каталоге

## Тестовые данные

| Поле | Значение |
|---|---|
| Название товара | КЕДЫ VANS |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить название товара
3. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Открывается новая страница в текущей вкладке, отображаемые товары содержат в названии "Vans" и тип товара "Кеды низкие" или "Кеды высокие". Результаты поиска отсортированы по популярности. Система обрабатывает "КЕДЫ VANS" так же, как "кеды vans" — результаты поиска в верхнем регистре идентичны результатам поиска в нижнем регистре.

## Фактический результат

Открылась новая страница в текущей вкладке. Показаны результаты поиска, содержащие "Vans" в названии, тип товара — "Кеды низкие" и "Кеды высокие". Полученные результаты отсортированы по популярности. Результаты поиска идентичны поиску в нижнем регистре.
