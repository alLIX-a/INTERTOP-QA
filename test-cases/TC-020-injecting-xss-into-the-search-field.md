# TC-021: Ввод XSS-инъекции в поле поиска товара

| Поле | Значение |
|---|---|
| **ID** | TC-021 |
| **Название** | Ввод XSS-инъекции в поле поиска товара |
| **Статус** | Пройдено |

## Предусловия

1. Главная страница открыта
2. Статус авторизации не имеет значения

## Тестовые данные

| Поле | Значение |
|---|---|
| XSS-инъекция | `<script>alert(1)</script>` |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить XSS-инъекцию
3. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Открывается новая страница в текущей вкладке, показывается сообщение "По вашему запросу ничего не найдено. Попытайтесь изменить запрос или выбрать другой раздел для уточнения". Товаров по запросу не найдено, alert показано не было.

## Фактический результат

Открылась новая страница в текущей вкладке, ожидаемое сообщение отобразилось. Товаров по запросу не найдено, alert не был выполнен.
