# TC-035: Проверка корректности итоговой стоимости товаров после удаления товара с корзины

| Поле | Значение |
|---|---|
| **ID** | TC-035 |
| **Название** | Проверка корректности итоговой стоимости товаров после удаления товара с корзины |

## Предусловия

1. Пользователь находится на любой странице сайта
2. Пользователь может быть авторизован или нет
3. В корзину добавлено хотя бы 2 товара (см. TC-030)

## Тестовые данные

| Параметр | Значение |
|---|---|
| Товар | 2 любых товара, добавленных в корзину |

## Шаги воспроизведения

1. Открыть раздел "Корзина", нажав на иконку корзины в правом верхнем углу страницы
2. Нажать на иконку мусорного ведра напротив любого из товаров в корзине

## Ожидаемый результат

Появляется окно с сообщением "Товар удален из корзины" с запущенным счётчиком в 3 секунды. По истечению 3 секунд товар пропадает из корзины. Итоговая стоимость товаров, указанная напротив "Стоимость товаров", уменьшается на стоимость удалённого товара.

## Фактический результат

Окно с сообщением появилось, счётчик запустился. По истечению 3 секунд товар пропал из корзины, итоговая стоимость была перерассчитана корректно.

## Статус

**Пройден**
