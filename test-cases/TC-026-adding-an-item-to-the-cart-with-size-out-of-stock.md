# TC-026: Добавление товара с отсутствующим в наличии размером

| Поле | Значение |
|---|---|
| **ID** | TC-026 |
| **Название** | Добавление товара с отсутствующим в наличии размером |

## Предусловия

1. Страница товара открыта
2. Пользователь может быть авторизован или нет
3. Товар имеет хотя бы один размер, отсутствующий в наличии

## Тестовые данные

| Параметр | Значение |
|---|---|
| Название товара | Кеды Vans |
| Цвет | Черный (выбран системой автоматически) |
| Размер | Любой из отсутствующих в наличии (отсутствующие в наличии размеры выделены белым цветом с серой рамкой вокруг) |
| Количество | 1 (установлен системой автоматически) |

## Шаги воспроизведения

1. Выбрать любой из размеров, отсутствующий в наличии

## Ожидаемый результат

Кнопка "Добавить в корзину" пропадает, появляются кнопки "Уведомить о наличии" и "Добавить в избранное".

## Фактический результат

Кнопка "Добавить в корзину" пропала, кнопки "Уведомить о наличии" и "Добавить в избранное" появились.

## Статус

**Пройден**
