# TC-074: Оформление заказа с экспресс доставкой + оплата при получении

| Поле | Значение |
|---|---|
| **ID** | TC-074 |
| **Название** | Оформление заказа с экспресс доставкой + оплата при получении |

## Предусловия

1. Пользователь зарегистрирован (см. ТС-001)
2. Пользователь авторизован (см. ТС-002)
3. В корзину добавлен хотя бы 1 товар (см. ТС-023)
4. Пользователь находится на любой странице веб-сайта

## Тестовые данные

1. Город: Алматы
2. Способ доставки: Экспресс доставка
3. Способ оплаты: Оплата при получении
4. Контактные данные получателя: подставляются из данных профиля автоматически

## Шаги воспроизведения

1. Открыть раздел «Корзина», нажав на иконку корзины в правом верхнем углу
2. В открывшемся окне корзины нажать на кнопку «Оформить заказ»
3. На открывшейся странице оформления заказа нажать на выпадающий список городов в разделе «Доставка»
4. В выпадающем списке найти и выбрать город, указанный в тестовых данных
5. В разделе «Доставка» выбрать способ доставки, указанный в тестовых данных

## Ожидаемый результат

Способ оплаты «Оплата при получении» пропадает из списка доступных способов оплаты в разделе «Оплата». Пользователь не может оформить заказ с оплатой при получении, выбрав экспресс доставку.

## Фактический результат

«Оплата при получении» пропала из списка доступных способов оплаты. Заказ не оформился.

## Статус

**Пройден**
