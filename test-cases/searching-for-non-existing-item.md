# TC-014: Поиск несуществующего товара

| Поле | Значение |
|---|---|
| **ID** | TC-014 |
| **Название** | Поиск несуществующего товара |
| **Статус** | Пройдено |

## Предусловия

1. Главная страница открыта
2. Статус авторизации не имеет значения
3. Товар "несуществующий товар" не существует

## Тестовые данные

| Поле | Значение |
|---|---|
| Название товара | несуществующий товар |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить название товара
3. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Открывается новая страница в текущей вкладке, показывается сообщение "По вашему запросу ничего не найдено. Попытайтесь изменить запрос или выбрать другой раздел для уточнения". Товаров по запросу не найдено.

## Фактический результат

Открылась новая страница в текущей вкладке, ожидаемое сообщение отобразилось. Товаров по запросу не найдено.
