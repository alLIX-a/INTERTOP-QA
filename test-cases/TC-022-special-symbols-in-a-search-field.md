# TC-022: Ввод спецсимволов в поле поиска товара

| Поле | Значение |
|---|---|
| **ID** | TC-022 |
| **Название** | Ввод спецсимволов в поле поиска товара |

## Предусловия

1. Главная страница открыта
2. Статус авторизации не имеет значения

## Тестовые данные

| Поле | Значение |
|---|---|
| Спецсимволы | `@#$%^&*()` |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить спецсимволы
3. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Открывается новая страница в текущей вкладке, показывается сообщение "По вашему запросу ничего не найдено. Попытайтесь изменить запрос или выбрать другой раздел для уточнения". Товаров по запросу не найдено, ошибок нет.

## Фактический результат

Открылась новая страница в текущей вкладке, ожидаемое сообщение отобразилось. Товаров по запросу не найдено, ошибок нет.

## Статус

**Пройден**
