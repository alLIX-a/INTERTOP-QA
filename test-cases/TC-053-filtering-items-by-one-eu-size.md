# TC-053: Фильтрация товаров по одному европейскому размеру

| Поле | Значение |
|---|---|
| **ID** | TC-053 |
| **Название** | Фильтрация товаров по одному европейскому размеру |

## Предусловия

1. Страница товаров открыта
2. Пользователь может быть авторизован или нет

## Тестовые данные

| Параметр | Значение |
|---|---|
| Европейский размер | 36 |

## Шаги воспроизведения

1. Нажать на кнопку "Размер одежды"
2. В выпадающем списке размеров нажать на кнопку "Европейские"
3. Выбрать размер 36

## Ожидаемый результат

Отображаются все и только товары размера 36 (европейский).

## Фактический результат

Отобразились все и только товары размера 36 (европейский).

## Статус

**Пройден**
