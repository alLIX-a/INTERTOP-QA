# TC-018: Поиск товара с запросом минимально допустимой длины (3 символа)

| Поле | Значение |
|---|---|
| **ID** | TC-018 |
| **Название** | Поиск товара с запросом минимально допустимой длины (3 символа) |

## Предусловия

1. Главная страница открыта
2. Пользователь может быть авторизован или нет

## Тестовые данные

| Поле | Значение |
|---|---|
| Название товара | van |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить название товара
3. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Открывается новая страница в текущей вкладке, показывая результаты, в названии которых содержится "van". Результаты отсортированы по популярности.

## Фактический результат

Открылась новая страница в текущей вкладке, результаты, в названии которых содержится "van", отображены и отсортированы по популярности.

## Статус

**Пройден**
