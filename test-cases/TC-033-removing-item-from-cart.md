# TC-033: Удаление товара из корзины

| Поле | Значение |
|---|---|
| **ID** | TC-033 |
| **Название** | Удаление товара из корзины |

## Предусловия

1. Пользователь находится на любой странице сайта
2. Пользователь может быть авторизован или нет
3. В корзину добавлен хотя бы 1 товар (см. TC-023)

## Тестовые данные

| Параметр | Значение |
|---|---|
| Товар | Любой товар, добавленный в корзину |

## Шаги воспроизведения

1. Открыть раздел "Корзина", нажав на иконку корзины в правом верхнем углу страницы
2. Нажать на иконку мусорного ведра напротив любого из товаров в корзине

## Ожидаемый результат

Появляется окно с сообщением "Товар удален из корзины" с запущенным счётчиком в 3 секунды. По истечению 3 секунд товар пропадает из корзины.

## Фактический результат

Появилось ожидаемое сообщение, таймер в 3 секунды был запущен. По истечению таймера товар был удалён из корзины.

## Статус

**Пройден**
