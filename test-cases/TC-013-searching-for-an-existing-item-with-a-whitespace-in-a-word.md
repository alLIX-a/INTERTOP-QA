# TC-013: Поиск существующего товара с пробелом внутри слова в названии товара

| Поле | Значение |
|---|---|
| **ID** | TC-013 |
| **Название** | Поиск существующего товара с пробелом внутри слова в названии товара |
| **Статус** | Пройдено |

## Предусловия

1. Главная страница открыта
2. Статус авторизации не имеет значения
3. Кеды бренда "Vans" существуют в каталоге

## Тестовые данные

| Поле | Значение |
|---|---|
| Название товара | ке ды vans |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить название товара
3. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Открывается новая страница в текущей вкладке, отображаемые товары содержат в названии "Vans" и тип товара "Кеды низкие" или "Кеды высокие". Результаты поиска отсортированы по популярности. Система обрабатывает "ке ды vans" так же, как "кеды vans" — результаты поиска в нижнем регистре с пробелами в поисковом запросе идентичны результатам при поиске в нижнем регистре без пробелов.

## Фактический результат

Открылась новая страница в текущей вкладке. Показаны результаты поиска, содержащие "Vans" в названии, тип товара — "Кеды низкие" и "Кеды высокие". Полученные результаты отсортированы по популярности. Результаты поиска идентичны поиску в нижнем регистре без пробелов.
