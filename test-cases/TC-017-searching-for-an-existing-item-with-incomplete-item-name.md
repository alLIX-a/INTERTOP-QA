# TC-017: Поиск существующего товара с неполным названием

| Поле | Значение |
|---|---|
| **ID** | TC-017 |
| **Название** | Поиск существующего товара с неполным названием |

## Предусловия

1. Главная страница открыта
2. Пользователь может быть авторизован или нет
3. Товары с "adi" в названии существуют в каталоге

## Тестовые данные

| Поле | Значение |
|---|---|
| Название товара | adi |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. В открывшемся поле ввода вставить название товара
3. Нажать на иконку лупы внутри поля поиска

## Ожидаемый результат

Открывается новая страница в текущей вкладке. Показываются все результаты, включающие в названии товара слово "adi". Результаты поиска отсортированы по популярности.

## Фактический результат

Открылась новая страница в текущей вкладке. Товары, название которых включает в себя "adi", показаны и отсортированы по популярности.

## Статус

**Пройден**
