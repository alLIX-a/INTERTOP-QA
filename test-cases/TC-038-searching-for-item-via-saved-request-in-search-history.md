# TC-038: Поиск товара по сохранённому запросу в истории поиска

| Поле | Значение |
|---|---|
| **ID** | TC-038 |
| **Название** | Поиск товара по сохранённому запросу в истории поиска |

## Предусловия

1. Пользователь находится на любой странице веб-сайта
2. Пользователь может быть авторизован или нет
3. Поиск товара "кеды vans" выполнен (см. TC-011)

## Тестовые данные

| Параметр | Значение |
|---|---|
| Товар | Осуществлённый ранее поисковой запрос "кеды vans" |

## Шаги воспроизведения

1. Нажать на кнопку "Поиск модных находок"
2. Стереть ранее осуществлённый поисковой запрос в поле поиска, нажав на иконку крестика внутри поля
3. В появившемся разделе "История поиска" нажать на любой из сохранённых запросов в разделе

## Ожидаемый результат

Открывается новая страница в текущей вкладке, показываются результаты, содержащие в названии "кеды vans". Результаты отсортированы по популярности.

## Фактический результат

Открылась новая страница в текущей вкладке, результаты, содержащие в названии "кеды vans" были показаны и отсортированы по популярности. При нажатии на иконку крестика содержимое поля не стирается: пользователь может очистить поле только при помощи клавиатуры своего устройства.

## Статус

**Пройден**
